# Tech Leadership — Guia Completo

> **Objetivo deste guia:** Servir como referência completa sobre **Tech Leadership e competências Staff+** — fundamentos, visão técnica, influência organizacional, liderança cross-team, mentoria, business acumen, comunicação escrita e liderança em falhas, otimizada para uso como **base de conhecimento para assistentes de código (Copilot/AI)** e consulta humana.
>
> **Público-alvo:** Tech Leads, Staff Engineers, Principal Engineers e Engineering Managers que buscam desenvolver competências de liderança técnica.

---

## Sumário

| Documento | Descrição |
