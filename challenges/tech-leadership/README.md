# Zero to Hero — Tech Leadership Challenge

> **Programa de especialização progressiva** em Liderança Técnica, do **Senior Engineer** ao **Fellow**.
> Cenários práticos, entregáveis concretos e rubrica de avaliação para cada nível da career ladder.

---

## 1. Resumo Executivo

Este programa desenvolve **competências de liderança técnica** através de cenários realistas, simulações, documentos estruturados e exercícios de reflexão. Cada nível mapeia diretamente uma posição na **engineering career ladder** — do Senior que lidera pelo exemplo no squad até o Fellow que define estratégia tecnológica de longo prazo.

**Domínio escolhido:** **TechNova** — uma empresa de tecnologia em crescimento acelerado (Série B → IPO) que enfrenta todos os desafios clássicos de scaling de pessoas, processos e cultura. O engenheiro assume progressivamente mais responsabilidades de liderança ao longo da jornada.

**Por que TechNova?**
- **Crescimento realista** — de 30 para 500+ engenheiros ao longo da trilha, espelhando desafios reais de scale-ups
- **Tensões naturais** — velocity vs quality, autonomy vs alignment, innovation vs stability
- **Contextos variados** — squads, chapters, tribes, áreas, plataforma, organização
- **Stakeholders diversos** — ICs, tech leads, EMs, PMs, designers, C-level, board
- **Cenários de crise** — incidents, layoffs, turnover, tech debt, deadline pressure
- **Decisões reais** — hiring, firing, promotions, reorgs, tech migrations, build vs buy

**O que NÃO é este programa:**
- Não é sobre management/gestão de pessoas (people management) — é sobre **liderança técnica**
- Não substitui um IC track — complementa com as soft skills necessárias em cada nível
- Não é teoria — cada nível tem **entregáveis concretos** avaliáveis

## 3. Competências por Nível

| Nível | Pessoas & Equipe | Conflito & Negociação | Cultura & Dinâmicas | Carreira & Desenvolvimento |
|-------|------------------|-----------------------|---------------------|---------------------------|
| **Senior** | Mentoria 1:1, pair programming, feedback direto | Divergências técnicas no squad, mediação em code review | Contribuir para cultura do squad, onboarding de novos membros | Auto-desenvolvimento, plano de carreira pessoal |
| **Staff** | Coaching de seniors, facilitação cross-team | Alinhamento técnico entre times, gestão de trade-offs | Estabelecer práticas entre times, guilds/chapters | Coaching de carreira, sponsorship |
| **Principal** | Talent assessment, hiring bar, entrevistas técnicas | Decisões técnicas controversas, stakeholder negotiation | Engineering practices de área, tech radar local | Talent development, succession planning |
| **Distinguished** | Org design, engineering ladder, calibration | Mediação org-level, vendor negotiations, crises | Cultura de engenharia organizacional, D&I | Leadership development program, mentoria de principals |
| **Fellow** | Advisory role, external mentorship, academic partnerships | Board-level negotiations, industry standards | Industry thought leadership, open source governance | Legacy building, next-gen leadership |

## 6. Formato dos Desafios

Cada desafio de soft skill segue uma estrutura consistente:

| Elemento | Descrição |
|----------|-----------|
| **Cenário** | Situação realista na TechNova com contexto detalhado |
| **Desafio** | Tarefa específica com requisitos claros |
| **Entregáveis** | Documentos, templates, scripts, análises — artefatos concretos |
| **Critérios de Aceite** | Checklist objetivo e verificável |
| **Framework de Referência** | Modelo teórico para fundamentar a prática |
| **Rubrica de Avaliação** | Auto-avaliação estruturada com 4 níveis (Iniciante → Expert) |

**Tipos de entregáveis:**
- Documentos estruturados (1:1 templates, feedback scripts, RFCs, ADRs)
- Análises escritas de cenários (trade-offs, decisões, post-mortems)
- Planos e frameworks (hiring plans, onboarding guides, career ladders)
- Reflexões pessoais (journaling estruturado com prompts específicos)
- Simulações documentadas (role-play scripts com rubrica)

## 7. Referências e Frameworks

| Framework | Aplicação na Trilha |
|-----------|-------------------|
| **SBI (Situation-Behavior-Impact)** | Feedback estruturado em todos os níveis |
| **GROW (Goal-Reality-Options-Will)** | Coaching e 1:1s (Senior → Staff) |
| **RACI** | Definição de responsabilidades (Staff → Principal) |
| **Radical Candor** | Feedback direto e empático (Senior) |
| **Staff Engineer Archetypes** (Larson) | Auto-diagnóstico de estilo de liderança (Staff) |
| **Team Topologies** | Organização de times (Principal → Distinguished) |
| **Westrum Organizational Culture** | Diagnóstico cultural (Distinguished) |
| **Wardley Mapping** | Estratégia tecnológica (Fellow) |
| **Technology Radar** (ThoughtWorks) | Gestão de portfólio técnico (Principal → Fellow) |
| **DORA Metrics** | Medição de engineering effectiveness (Staff → Distinguished) |
